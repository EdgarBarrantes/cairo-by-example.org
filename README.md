# [Cairo by Example](https://cairo-by-example.org)

Learn Cairo and StarkNet Smart Contract programming with example code.

## Setup

1. `git clone <url>`
2. `pnpm install`
3. `cp .env.example .env.local`
4. `cp .env.example .env.production`
5. Update `.env` files
6. `pnpm run dev`

## Useful Commands

```sh
pnpm install

pnpm run dev

pnpm run build

pnpm run start

pnpm run lint

pnpm run format

pnpx vercel [--prod] [--force]
```
